> Expanding our storage offerings allows more users to engage with our app.

## Key Achievements

### Secure Integration

We integrated sign on and storage calls to allow for secure handoff to mobile developers, enabling CSi users to sign in and store their schematics seamlessly. The solution included secure single sign on (SSO) functionality.

### AWS S3 Integration

Built a robust integration with Amazon S3 to provide scalable, reliable storage for user schematics and documents. The integration ensures secure access and efficient data management.

### Development Workflow

We introduced their development team to Lando to aid and easily spin up development environments, improving their development workflow and productivity.

## Technologies

- Drupal 7 backend
- AWS S3 for storage
- Mobile app integration
- Secure single sign on
