# Switchback Solutions

A modern Astro website for Switchback Solutions, a consultancy that creates digital solutions for land conservancies and trail organizations.

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:4321`

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Design System

### Colors

- **Primary (Evergreen)**: `#2E5D47`
- **Secondary (Warm Sand)**: `#E7D9C4`
- **Accent (Sunset Amber)**: `#E6A85B`
- **Dark Slate**: `#1F1F1F`
- **Mist White**: `#F8F8F7`

- **Headings**: Raleway
